# Get My Fixings
- Community curated recipe listing web application

# Features
- Recipes
  - Listing (pick)
  - Search by type
  - Rate
- Comments
  - Post comment
  - Delete comment
  - List comments for recipe
- Lists
  - Generate
  - Refresh
  - Delete

# Site flow
- Guest
  - View recipe index
  - View single recipe pages
  - Access Login page
  - Access Register page
- User
  - View recipe index
  - View single recipe pages
  - Rate recipe
  - Comment on recipe
  - View lists index
  - Generate list
  - Refresh list
  - Delete list
- Admin
  - All user flow/privileges
  - Create recipe
  - Delete owned recipes
  - Add ingredients to owned recipes
  - View users index
  - Delete users
